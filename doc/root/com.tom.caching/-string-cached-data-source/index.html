String Cached Data Source

abstract class StringCachedDataSource<ID>(val cache: Cache<ID, String>) : CachedDataSource<ID, String>

Simple implementation of CachedDataSource that handles general caching String data using a generic Identifier of type ID

The implementer of this class needs to only implement the fetchSupplier method which can often be done in one line. It must return the freshly fetched String that corresponds to the given String identifier
